# Brincando com stack e heap

Este trabalho foi feito no dia 5 do minicurso de C. O objetivo é mostrar as diferenças entre a pilha de execução e o heap
Além disso, foi mostrado também uma pequena apresentação sobre aritmética de ponteiros, passagem por referência e ponteiros para função
# Como construir
```bash
make
# O binário está disponível em build/debug{.exe}
```

# Conhecimentos utilizados
### Dia 4 - Ponteiros, compiladores
### Dia 5 - Structs, Unions, ponteiro pra função e outros transformers
### Dia 6 - Pré-processadores e macros
